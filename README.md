# Smooth-Menu-Bar

A modern, sleek, and highly interactive Bottom Navigation Bar built with HTML, CSS, and JavaScript. This menu features a "floating" active indicator with a smooth liquid-like animation that transitions seamlessly between menu items.

# The menu includes the following navigation points:

Home: The default landing state.
Profile: User account settings.
Messages: Chat and notifications.
Camera: Quick access to photo features.
Settings: App configuration and tools.
